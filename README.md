# Colessionassim - Refatoração Responsiva com Bootstrap


Este projeto consiste da home-page "Colessionassim", uma plataforma voltada para o mercado de colecionadores. O objetivo principal desta etapa foi evoluir o layout original, construído com CSS puro, para uma estrutura moderna e responsiva utilizando o framework **Bootstrap 5**.


 **HTML5:** Estruturaçãodo conteúdo (`<header>`, `<main>`, `<section>`, `<footer>`).
 **CSS3:** Estilização customizada (cores específicas, tamanhos fixos de imagens com `object-fit`, efeitos de `hover` e transições suaves).
**Bootstrap 5:** Framework principal para o sistema de grids responsivos (`col-12`, `col-md-4`), barra de navegação, espaçamentos utilitários.
 **Git/GitHub:** Controle de versão com commits descritivos e tags de versão (v2.0).


**Cabeçalho:** Navbar responsiva com campo de pesquisa centralizado e menu colapsável (hambúrguer) para mobile. Fundo estilizado com a cor de destaque (Amarelo).
**Seção Principal:** Banner fluido e texto introdutório chamativo.
**Seção Secundária (Cards):** Utilização do sistema de Grid do Bootstrap para alinhar os cards das categorias de colecionáveis, com imagens redimensionadas e padronizadas via CSS para manter a consistência visual.
**Rodapé:** Assinatura do projeto minimalista e fixada ao fundo.
**Interatividade:** Efeito de "lift" (elevação) suave ao passar o mouse sobre os cards e links, garantindo uma melhor experiência do usuário (UX).
